All photoplot files are in inches, 4.4 format.

The following extensions are used to describe 
the various photoplot/NC drill files:

.AP     APerture Definition and Statistics
.TOP    TOP copper layer
.BOT    BOTtom copper layer
.GND    GouND plane
.PWR    PoWeR plane
.ML1    Middle Layer 1
.ML2    Middle Layer 2
.ML3    Middle Layer 3
.ML4    Middle Layer 4
.ML5    Middle Layer 5
.ML6    Middle Layer 6
.ML7    Middle Layer 7
.ML8    Middle Layer 8
.TSM    Top Solder Mask
.BSM    Bottom Solder Mask
.TSS    Top Silk Screen
.BSS    Bottom Silk Screen
.TSP    Top Solder Paste     REMOVE 0.050 DIA ROUND PADS (FUDICIALS)
.BSP    Bottom Solder Paste  REMOVE 0.050 DIA ROUND PADS (FUDICIALS)
.DWG    drill DraWinG (Fabrication Drawing) PHOTOPLOT FILE
.NCD    N/C Drill file, Inches, ASCII none, Trailing Zero Suppression
.TAS    Top Assembly Drawing
.BAS    Bottom Assembly Drawing
